# <i class="fa fa-terminal"></i> 使用Fisher

如果您使用的是Linux或Mac OS X，[fisher](https://github.com/jorgebucaran/fisher)是管理[fish shell](http://fishshell.com/)設定的好工具。

JHipster開發團隊中的一些人在Fisher上使用fish shell，如果您看到人們在其終端上使用快捷方式，那麼魔術就來自這裡！

## Fisher JHipster外掛

JHipster Fisher外掛可從GitHub上的[https://github.com/jhipster/jhipster-fisher-plugin](https://github.com/jhipster/jhipster-fisher-plugin)獲得。

目前，它僅新增快捷方式（此處為[完整](https://github.com/jhipster/jhipster-fisher-plugin/blob/main/conf.d/jhipster.aliases.fish)清單），但我們歡迎您提供更好的自動完成功能！

1. Install it via ``fisher`` command:

   `fisher add jhipster/jhipster-fisher-plugin`

有關[Fisher](https://github.com/jorgebucaran/fisher)的更多詳細訊息，請參見其[用法](https://github.com/jorgebucaran/fisher#usage)部分。
