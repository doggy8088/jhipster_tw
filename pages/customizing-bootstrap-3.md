# <i class="fa fa-css3"></i> 自定義Bootstrap 3

我們對Angular和React都提供了強大的支援，它們都是更現代的工具，希望您會喜歡它們！

如果仍然需要使用AngularJS 1.x，請看一下 [我們的文件](https://www.jhipster.tech/documentation-archive/).
