# <i class="fa fa-css3"></i> 自定义Bootstrap 3

从JHipster 5开始，JHipster不再支持AngularJS和Bootstrap 3。

我们对Angular和React都提供了强大的支持，它们都是更现代的工具，希望您会喜欢它们！

如果仍然需要使用AngularJS 1.x，请看一下 [我们的文档]({{ site.url }}/documentation-archive/).
