## JHipster 快速开始

1. 安装 [Java 17](https://adoptium.net/temurin/releases/?version=17)、[Node.js](http://nodejs.org/)、[Git](http://git-scm.com/)（推荐）和 [Docker](docker-compose/)（推荐）
2. 安装 JHipster `npm install -g generator-jhipster`
3. 创建新目录并进入 `mkdir myApp && cd myApp`
4. 运行 JHipster 并按照屏幕提示操作 `jhipster`
5. 使用 [JDL Studio](https://start.jhipster.tech/jdl-studio/) 建模您的实体并下载生成的 `jhipster-jdl.jdl` 文件
6. 使用 `jhipster import-jdl jhipster-jdl.jdl` 生成您的实体
